# T66 Docs

Documentation is organized by function instead of leaving project notes in the Unreal project root.

- `Audit/` contains audits and deep inspections.
- `Implementation Plan/` contains detailed execution packs that split larger features into agent-specific ownership and handoff instructions.
- `Plans/` contains implementation and rollout plans.
- `MASTER DOCS/` contains active project policy and long-lived master references outside the dedicated anti-cheat folder.
- `ANTI_CHEAT/` contains the active anti-cheat policy, integrity, moderation, and implementation references.
- `Systems/` contains current system notes, handoffs, and technical documentation.
- `Art/` contains art workflow and prompt-pack docs.
- `Archive/` contains quarantined or superseded material kept for reference, including historical system docs.
