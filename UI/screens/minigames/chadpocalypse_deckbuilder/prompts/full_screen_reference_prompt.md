# Chadpocalypse Deckbuilder Full-Screen Reference Prompts

Use case: `ui-mockup`
Asset type: simple full-screen 1920x1080 reference mockups
Status: simplified v2 prompt set. Generate only the two screens below.

Global rules:
- Generate raster pixel-art game UI references only.
- Keep layouts simple, readable, and close to a first playable screen.
- Do not generate reward screens, encounter maps, shops, deck preview drawers, or component sheets.
- Keep live runtime text, card rules, costs, and values as blank wells or simple placeholder bars.
- No watermark, no callout labels, no diagram arrows, no multi-screen grid.

## Prompt 01: Main Menu

Create a simple main menu mockup for the T66 minigame `Chadpocalypse Deckbuilder`.

Composition:
- 1920x1080 fullscreen.
- Large title area in the upper-left or upper-center.
- Dungeon descent visual with stairs, cards, or a relic pedestal.

- Generous empty space and a clear dungeon-deckbuilder mood.

Avoid:
- Combat state.
- Encounter maps.
- Shops.
- Reward drafts.
- Card text grids.
