T66 Map Assets
==============
Consolidated assets for the procedural landscape tool.

- Grass: uses in-project world grass assets (`/Game/World/Props/Grass` + `/Game/World/Props/Grass/Materials/Material_0_014`)
- Landscape, trees, rocks: from Cozy Nature (MI_Landscape, Grass_LayerInfo, SM_Tree1/2/3, MI_Tree1/2/3, SM_Rock1/2/3, MI_Rock1/2/3)

Run once: Window -> T66 Tools -> Setup T66 Map Assets
to copy the source assets here. Then "Generate Procedural Hills Landscape (Dev)" will use these copies.
