# Generated Dota UI Surfaces

These source-art plates were generated for the Dota-style UI pass and are intended to stay text-free so Slate can stretch them safely.

## Files

- `dota_primary_button_plate.png`
  - Use for large primary CTA buttons.
  - Intended treatment: green enamel center, dark metal frame, subtle bevel highlight.
  - Keep as a 9-slice or crop source, not as a final button with baked text.

- `dota_inventory_slot_frame.png`
  - Use for inventory, idol, or equipment socket framing.
  - Intended treatment: square recessed cavity with readable corners and a dark inner pocket.
  - Works as either a direct slot frame or as source for a cleaner hand-authored trim pass.

## Prompt Intent

- Match Dota-style fantasy MOBA chrome, not a full screen mockup.
- Preserve transparent backgrounds for direct UI compositing.
- Avoid text, logos, iconography, perspective distortion, and excessive ornament.
