# Settings Reference Pack

This folder stores the immutable and derived artifacts used to reconstruct `Settings`.

## Canonical Canvas

- layout master target: `1920x1080`
- hi-res companion target: `3840x2160`

## Expected Files

- `reference_layout.json`
- `asset_manifest.json`
- `debug/`
- `SpriteSheets/`
- sliced family outputs:
- `SheetSlices/TopBar/`
- `SheetSlices/Center/`
- `SheetSlices/LeftPanel/`
- `SheetSlices/RightPanel/`
- `SheetSlices/Decor/`
- promoted runtime assets:
- `TopBar/`
- `Center/`
- `LeftPanel/`
- `RightPanel/`
- `Decor/`

Keep the approved layout reference separate from generated family sheets and runtime-ready slices.
