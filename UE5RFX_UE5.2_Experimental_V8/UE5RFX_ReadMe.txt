Installation:

To use UE5RFX_UE5.2_Experimental in your Unreal Engine:

Copy the UE5RFX_UE5.2_Experimental_V8 folder into your Unreal Projects directory.

On Windows, this is typically located at:
C:\Users\<YourUsername>\Documents\Unreal Projects\

Open Unreal Engine 5.2 (or newer) and select the project from the launcher.

Upgrading to Newer Versions:

Since the project is entirely blueprint-based, upgrading to newer versions of Unreal Engine should work without any issues.


Feel free to report any problems if encountered during upgrades.
